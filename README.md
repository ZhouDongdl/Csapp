## 实验前准备  
由于所有实验均为linux环境下进行的，需安装Linux系统或者docker，这里选择VMware+Ubuntu的方式  
安装好Ubuntu后，在终端输入以下命令  
```
sudo apt-get install build-essential  
sudo apt-get install gcc-multilib  
sudo apt-get install gdb
```
